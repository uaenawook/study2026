## 函数的参数
#### 函数实参

    真是传给函数的参数
    可以是：常量、变量、表达式、函数等
#### 函数形参

    在函数名括号中的变量
    形参，只在函数调用时存在

## 函数的调用    
#### 传值调用

    函数的形参和实参分别占有不同的内存块
    对形参的修改不会影响实参
#### 传址调用

    1.传过去的是函数变量的地址传过去
    2.这种可以让函数内部外部，建立真正的关系，
        函数内部可以直接操作函数外部的变量。

#### 函数的递归：

**定义**

    在函数中直接或间接的调用自己
    能代替多次大量的重复 
    **大事化小**

**必要条件**

    1.必须要限制条件，不满足条件时，递归结束
    2.必须每次接近这个限制条件

**函数栈溢出**

    死递归 - 栈溢出 - stack overflow
    
    每一次函数调用都要在内存中开辟空间
    
    **函数栈帧的创建和销毁** 搜索教程

举例：
函数的递归  输入1234  打印1 2 3 4
    
    #include <stdio.h>
    
    void print(unsigned int n)
    {
    	if (n > 9)
    	{
    		print(n / 10);
    	}
    	printf("%d ", n % 10);
    }
    
    int main()
    {
    	unsigned int n = 0;
    	scanf("%u", &n);
    	print(n);
    	return 0;
    }

指针++
    
    指针++ 就是指针指向的地址+1 
    也就时指向了下一个元素 
    字符指针+1
    char* p + 1  向后跳一个字节
    整形指针+1
    int* p + 1   向后跳四个字节

什么时候用递归：
​    
    1.当解决一个问题递归和非递归都可以使用，未有明显问题，就用递归
    2.当解决一个问题递归简单，非的递归复杂，且递归没有明显问题，就用递归
    3.如果用递归简单，但有明显问题，就不用递归。
