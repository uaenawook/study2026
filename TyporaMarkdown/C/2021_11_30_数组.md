# 一维数组

    全局变量 char arr[5]  存的0
    局部变量 char arr1[5]  存的随机值
    
    数组使用下标访问元素 - arr[5]
    数组在内存中连续存放的 由低到高排列
#### 一维数组的创建

    1.  int arr1[10]; 指定元素大小 不初始化
    
    2.  int count = 10;
        int arr2[count]; 错误 [] 内必须为常量
    
    3.  char arr3[10];
        float arr4[1];
        double arr5[20];

#### 一维数组的初始化

    int arr1[10] = {1,2,3}; 
        不完全初始化：1 2 3 0 0 0 0 0 0 
    int arr2[] = {1,2,3,4};
        不指定元素大小初始化 1 2 3 4 \0
    int arr3[5] = {1，2，3，4，5}；
        指定元素大小完全初始化 1 2 3 4 5 
    char arr4[3] = {'a',98, 'c'};
        指定元素大小 完全初始化 a b c
        因为在内存中存的是 ASCII码值
    char arr5[] = {'a','b','c'};
        未指定元素大小初始化 a b c \0
    char arr6[] = "abcdef";
        未指定元素大小初始化 a  b  c  d   e   f  \0
                    ASCII值  97 98 99 100 101 102

一维数组初始化区别

    char arr1[] = "abc";  a b c \0
    char arr2[3] = {'a','b','c'}; a b c

#### 数组名是什么 

    1.数组名是首元素的地址
    例外：
        1.sizeof(数组名)，表示整个数组，所以计算的是整个数组的代销
        2.&数组名，这里表示的整个数组，取出的数组的地址
    
    int arr[10] = {1,2,3,4,5};
    printf("%p\n",arr);
    printf("%p\n",&arr[0]);
    printf("%p\n",&arr);
    值相同，意义不相同  
    printf("%p\n",arr + 1); 加4个字节
    printf("%p\n",&arr[0] + 1);  加4个字节
    printf("%p\n",&arr + 1); 加40个字节 因为数组有10个元素
