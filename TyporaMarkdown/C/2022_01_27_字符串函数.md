strcpy字符串拷贝

    strcpy(arr1,arr2)
    把arr2放在 arr1中包括\0
    \0结束 后面的不会管
    arr2必须有\0
    arr1必须足够大，必须可改变

strcat 字符串连接 字符串追加

    返回 目标空间的地址
    #include <string.h>
    char *strcat( char *strDestination, const char *strSource );
    
    char arr[20] = "abc";  
    strcat(arr1,"def");
    
    char arr[20] = "abc";
    char arr2[] = {'d','e','f','\0'};
    strcat(arr,arr2);
    注意：
        源字符串必须以 '\0' 结束。
        目标空间必须有足够的大，能容纳下源字符串的内容。
        目标空间必须可修改。
        字符串自己给自己追加，如何

strcmp  比较字符串内容
        
    返回 <0  =0  >0   arr1-arr2 
    <string.h>
    int strcmp ( const char * str1, const char * str2 );
    char arr1[] = "abcd";
    char arr2[] = "abc";
    strcmp(arr1,arr2);//1    0     -1

小结 
    
    strcpy
    strcat
    strcmp
    不安全 长度不受限制
    
    strncpy(arr1,arr2,2) arr2拷贝的长度
    strncat(arr1,arr2,3) arr追加的长度 结束后自动加上\0
        arr长度满足 则追加3个
        不满足，则有多少追加多少
    strncmp(arr1,arr2,4) 所比较的字符长度
    相对安全 长度受限制
