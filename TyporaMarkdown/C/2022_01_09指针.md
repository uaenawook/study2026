## 指针是什么

    计算机科学中， **指针**
    
    1.
    地址指向了确定的内存空间
    所以 -地址- 形象的被称为 -指针
    
    2.
    int a = 10l
    int * pa = &a
    pa是用来存储地址的 pa叫做指针变量



    32位机器：
        32根地址线（1/0）
        00000000000000000000000000000000
        ...
        11111111111111111111111111111111
        这样的二进制序列，如果作为内存单元的编号
        2的32次方个地址，能够管理2的32次方个内存单元
        
        一个序列号是32位 32bit  存成字节是 32/8 = 4byte
        指针是4个字节
        
    64位机器：
        一个序列号是64位 64bit  存成字节是 64/8 = 8byte
        指针是8个字节

**一个小的单元是（1字节）**

### 野指针

    造成野指针的原因
    
    1.指针未初始化
    int main()
    {
    	int* p;//未初始化 - 里面 放的随机值
    
    	*p = 20;//通过p中的随机值，找到一个空间，这个空间不属于我们当前的程序
    	//就造成了非法访问，p就是野指针
    	return 0;
    }
    
    2.指针越界访问
    int main()
    {
    	int arr[10] = { 0 };
    	int i = 0;
    	int* p = arr;
    	for (i = 0; i <= 10; i++)
    	{
    		*p = i;
    		p++;
    	}
    	return 0;
    }
    
    3.指针指向的空间释放
    int* test()
    {
    	int a = 10;
    	return &a;   
    	尽量避免返回 变量的地址
    }
    int main()
    {
    	int* p = test();
    	printf("%d\n", *p);
    	return 0;
    }
    p存放了test（）返回的地址
    此时test（）返回后释放了
    又去打印（访问）p存的地址  就造成了越界访问
    造成了野指针
    
    int* p2;
    *p2 = 100;
    错误
    避免
        if（p2 ！=NULL）
        {
            *p2 = 100；
        }

### 如何 避免野指针

##### 1.指针初始化

        int* p1 = &a //明确的初始化，确定指向
##### 2.小心指针越界


##### 3.指针指向的空间释放即使指针置NULL

        int* p2 = NULL //不知道指到哪里，初始化成NULL
##### 4.避免返回局部变量的地址

        return &a；
##### 5.指针都是用之前检查有效性

        int* p2;
        *p2 = 100;
        错误
        避免
            if（p2 ！=NULL）
            {
                *p2 = 100；
            }

    1.指针++的方法
    int  my_strlen1(char* arr)
    {
    	int count = 0;
    	while (*arr != '\0')
    	{
    		count++;
    		arr++;
    	}
    	return count;
    }
    
    2.指针 - 指针的方法
    int my_strlen2(char* arr)
    {
    	int* s = arr;
    	while (*arr != '\0')
    	{
    		arr++;
    	}
    	return arr - s;
    }

        2.&数组名 - 取得是数组的地址

    {
